# `Main`

Renders a `main` component.

## Usage

```jsx
import { Main } from 'components';

const MyCTA = () => (
  <Main>
    <div>Hello</div>
  </Main>
);
```

## Props

The `Main` component accepts the following props:

### className

An optional class name for the main component.

Type: `String`
Required: No
